# Sitema de Apoio a Calamidades Públicas "ConectaSolidária".
Guia Técnico e Explicação do Código: Sistema de Apoio a Calamidades Públicas "ConectaSolidária". O presente tema, pois tem como objetivo o desenvolvimento de uma solução tecnológica voltada para um problema real, que é a organização da arrecadação e distribuição de doações em situações de calamidade pública. Quanto aos procedimentos, o trabalho será desenvolvido por meio de pesquisa bibliográfica e documental, com base em livros, artigos científicos e legislações relacionadas à gestão de crises, logística humanitária e uso da tecnologia para fins sociais. Essa etapa tem como finalidade fornecer embasamento teórico para o desenvolvimento do sistema. O projeto terá como área de aplicação a região de Brasília e entorno, onde será proposta a centralização da arrecadação de doações, com posterior organização e distribuição para áreas afetadas por calamidades públicas. Essa delimitação geográfica permite tornar o sistema mais viável e direcionado à realidade local, contribuindo para maior eficiência na logística e no atendimento às necessidades da população. Este documento detalha a arquitetura e a lógica de programação por trás do sistema web da "ConectaSolidária". O foco é explicar como HTML, CSS, PHP e JavaScript interagem para criar as funcionalidades do site.
